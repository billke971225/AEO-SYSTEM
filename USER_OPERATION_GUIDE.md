# 用户操作指南 - AI SEO博客生成系统

## 🎯 **系统简介**

这是一个专门为**AI搜索引擎优化（AEO）**设计的多用户SaaS系统，帮助您的网站在ChatGPT等AI工具中获得更好的推荐排名。

---

## 🚀 **快速开始（5分钟上手）**

### **第1步: 注册登录**
```
1. 访问: http://your-domain.com/multi-user
2. 点击"注册"标签
3. 填写信息:
   - 姓名: 您的姓名
   - 公司: 您的公司名称
   - 邮箱: 您的邮箱
   - 密码: 设置密码
4. 点击"注册账户"
5. 检查邮箱验证链接
```

### **第2步: 配置API密钥**
```
1. 登录后点击"API密钥"标签
2. 添加必需的API密钥:
   
   OpenAI (必需):
   - 获取: https://platform.openai.com/api-keys
   - 格式: sk-...
   - 用途: ChatGPT测试、图片生成
   
   Google Gemini (推荐):
   - 获取: https://makersuite.google.com/app/apikey
   - 格式: AIza...
   - 用途: 成本优化、文化分析
   
   Nanebanaen (可选):
   - 用途: 低成本图片生成
   
   Cloudinary (可选):
   - 用途: 图片处理优化
```

### **第3步: 选择功能模块**
```
三大核心模块:

模块1: 自动话题生成
- 适合: 已有网站，想优化现有产品
- 输入: 网站URL
- 输出: AI优化的话题和博客

模块2: 市场扩展发现  
- 适合: 想发现新市场机会
- 输入: 产品描述
- 输出: 新场景和市场机会

模块3: 节假日自动化
- 适合: 想抓住节假日营销
- 输入: 产品类型
- 输出: 自动化节假日内容
```

## 📋 **详细操作流程**

### **模块1: 自动话题生成系统**

#### **步骤1: 网站分析**
```
1. 进入"自动话题生成"页面
2. 输入您的网站URL: https://your-website.com
3. 点击"开始分析"
4. 系统自动分析:
   - 网站内容和产品信息
   - 目标用户群体
   - 核心价值主张
   - SEO关键词
```

#### **步骤2: 竞争对手发现**
```
系统自动执行:
1. 识别同类竞争网站
2. 分析竞争对手策略
3. 发现市场空白点
4. 生成竞争分析报告

您可以查看:
- 主要竞争对手列表
- 竞争对手优势分析
- 市场定位对比
```

#### **步骤3: 痛点爬取**
```
系统自动从多个平台爬取用户痛点:
- Trustpilot 用户评价
- Google Reviews 评论
- Reddit 讨论
- Twitter 用户反馈

AI智能分析:
- 痛点分类和优先级
- 情感分析
- 频率统计
- 解决方案建议
```

#### **步骤4: 话题生成**
```
基于痛点数据自动生成话题:
1. 查看生成的话题列表
2. 每个话题显示:
   - 话题标题
   - 相关痛点
   - 预期效果评分
   - AI推荐概率
3. 您可以:
   - 选择感兴趣的话题
   - 修改话题内容
   - 添加自定义话题
```

#### **步骤5: ChatGPT真实测试**
```
系统使用您的OpenAI API密钥进行真实测试:
1. 自动生成用户问题
2. 调用ChatGPT API获取回答
3. 分析您的网站是否被推荐
4. 检查推荐排名位置
5. 分析推荐原因

测试结果显示:
✅ 通过: 网站被ChatGPT推荐
❌ 未通过: 网站未被推荐
📊 排名: 推荐位置（第1、2、3...）
```
